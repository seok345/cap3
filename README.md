# 📌 1. 서론

모바일 MOBA 장르는 전 세계적으로 가장 경쟁이 치열한 장르 중 하나이며,  
그 중심에는 **Wild Rift(라이엇 게임즈)**와 **Mobile Legends(몽톤/Moonton)**가 존재한다.

이 두 게임은 각각 다른 전략·운영 방식·경제 시스템을 가지고 있으며,  
유저 경험(UX), 게임 운영, 밸런스 정책 등에서 뚜렷한 차이를 보인다.

본 프로젝트는 **Google Play Store 리뷰 80,000건 이상**을 수집하여  
두 게임의 실제 사용자 반응을 **정량적으로 비교**하는 분석을 수행했다.

### 🎯 분석 목표

- 두 게임의 **장기적 감성 변화(긍정/부정 비율)** 비교  
- **감성 역전 시점 감지** 및 해당 시점에 게임 내 발생한 이벤트/패치 분석  
- LDA 기반 **토픽 모델링을 통한 핵심 불만·칭찬 요소 도출**  
- 리뷰 데이터를 기반으로 한 **게임 운영 품질 평가**

즉, 본 프로젝트는 단순한 데이터 분석을 넘어  
게임 운영 관점에서의 통찰(Insight)을 제공하는 것을 목표로 한다.

---

# 📌 2. 데이터 수집 및 구조

Google Play Scraper를 활용하여 안정적으로 리뷰 데이터를 수집했다.  
Play Store는 API 제공이 없기 때문에 Selenium 방식보다 빠르고 차단 위험이 적다.

수집된 데이터 예시는 다음과 같다:

```
mobile_legends_reviews_total_40000.csv
wild_rift_reviews_total_40000.csv
```

데이터 컬럼 구성:

- **userName**: 리뷰 작성자  
- **content**: 리뷰 텍스트  
- **score**: 별점  
- **at**: 작성 날짜  
- **app_name**: Wild Rift / Mobile Legends 구분  

---

# 📌 3. 감성 라벨링

감성 분석을 위해 별점 기반 rule-based 라벨링을 수행했다.

| 별점 | 라벨 | 의미 |
|------|------|------|
| 4~5 | 1 | 긍정 리뷰 |
| 1~2 | 0 | 부정 리뷰 |
| 3 | 제거 | 중립 → 감성 학습에 불필요 |

중립(3점) 리뷰는 긍·부정 판단이 어려워 제거한다.  
이는 텍스트 감성 모델의 학습 정확도를 높이는 데 크게 기여한다.

---

# 📌 4. EDA – 데이터 전처리 및 분포

리뷰 데이터는 원본 그대로 사용하기 어렵기 때문에  
다양한 전처리 작업을 수행했다.

- 불용어 제거  
- 반복 문자 축소 (예: "ㅋㅋㅋㅋ" → "ㅋㅋ")  
- 띄어쓰기 오류 정규화  
- 중복 리뷰 제거  
- 게임별 데이터량 imbalance 문제 해결(undersampling 적용)

### 📊 분석 요약

- Wild Rift는 초기에는 긍정 비율이 낮게 시작했으나  
  **지속적인 패치 및 품질 개선으로 점진적 상승**  
- Mobile Legends는 높은 초기 평가를 받았지만  
  **트롤·밸런스·과금 요소**로 인해 장기적 하락세

### 🔥 감성 "역전 시점" 자동 탐지

| 역전 번호 | 시기 | 설명 |
|-----------|--------|--------|
| 1차 역전 | **2021년 12월** | MLBB 부정 리뷰 급증(매칭/트롤 문제) |
| 2차 역전 | **2024년 4월** | MLBB 밸런스 논란 & Wild Rift 안정성 개선 |

그래프 상에서도 해당 구간에 자동으로 **역전 마커(빨간 표시)**가 추가되어  
두 게임의 감성 흐름 변화를 직관적으로 이해할 수 있다.

# 📌 7. 역전 시기 이슈 분석 (게임 내부 이벤트 기반)

## 🟦 1차 역전 (2021년 12월 ~ 2022년 3월)

### ✔ Wild Rift
- 2022년 3월 패치 3.1: 대규모 시스템 리워크  
- 새로운 콘텐츠 제공  
- 서버 안정화 개선  
→ 긍정 감성 급증

### ✔ Mobile Legends
- ‘트롤 증가’ 리뷰 급증  
- 밸런스 붕괴 논란  
→ 부정 감성 증가

---

## 🟧 2차 역전 (2024년 4월)

### ✔ Wild Rift
- 대규모 패치는 없지만  
  ‘서버 지연 개선’과 같은 사소한 품질 안정이 긍정적으로 작용  

### ✔ Mobile Legends
- 2024년 리뷰에서  
  **“밸런스 논란”“트롤 증가”“과금 유도 강화”**가 반복 언급  
→ 유저 감성이 급격히 악화

# 📌 9. 최종 결론

## ✔ Wild Rift
- **패치·콘텐츠 업데이트 시점과 감성 상승이 명확히 연동**
- 매칭 시스템 불만은 여전하지만  
  그래픽·연출·운영 신뢰도 측면에서 안정적 상승

## ✔ Mobile Legends
- **영웅 다양성·스킨 퀄리티는 강점**  
- 하지만 과금·매칭·트롤 관련 구조적 문제로  
  장기적으로 감성 비율 하락

## ✔ 종합 결론
- Wild Rift는 **운영 안정성 + 신규 콘텐츠의 지속 제공**으로  
  2022년 이후 장기 우세 흐름을 유지  
- Mobile Legends는 **과금·트롤 문제 개선 없이는 감성 회복 어려움**
